# DENISE Interposer - KiCad Design

PCB interposer that sits between the Commodore Amiga's Super DENISE (8373) chip and its motherboard socket. Passively taps 16 video/sync signals and level-shifts them from 5V TTL to 3.3V LVCMOS for the Tang Primer 20K FPGA.

## Schematic Generation

The schematic is generated by a Python script for maintainability:

```bash
cd kicad/
python3 generate_schematic.py
```

This creates `denise_interposer.kicad_sch`. Open the `.kicad_pro` project file in KiCad 8.

## Circuit Description

### Pass-Through
All 48 DENISE DIP pins pass straight through from top socket (where DENISE plugs in) to bottom pins (which plug into the Amiga motherboard socket). The interposer is electrically invisible to the Amiga.

### Signal Taps
16 signals are tapped from the pass-through and routed to the level shifters:

| Signal Group | DENISE Pins | Level Shifter |
|-------------|-------------|---------------|
| R0-R3 (Red) | 20-23 | U1 (A1-A4) |
| B0-B3 (Blue) | 24-27 | U1 (A5-A8) |
| G0-G3 (Green) | 28-31 | U2 (A1-A4) |
| /CSYNC | 32 | U2 (A5) |
| /ZD | 33 | U2 (A6) |
| CDAC | 34 | U2 (A7) |
| 7M | 35 | U2 (A8) |

### Level Shifting
Two 74LVC245 ICs convert 5V TTL to 3.3V LVCMOS:
- Powered at 3.3V (inputs are 5V tolerant)
- DIR pin tied LOW (A-to-B direction)
- /OE pin tied LOW (always enabled)
- Each IC has a 100nF bypass capacitor

### Output Connector
J2 is a 2x10 pin header (2.54mm pitch) connecting to the FPGA:

| Pin | Signal | Pin | Signal |
|-----|--------|-----|--------|
| 1 | R0 | 2 | R1 |
| 3 | R2 | 4 | R3 |
| 5 | B0 | 6 | B1 |
| 7 | B2 | 8 | B3 |
| 9 | G0 | 10 | G1 |
| 11 | G2 | 12 | G3 |
| 13 | /CSYNC | 14 | /ZD |
| 15 | CDAC | 16 | 7M |
| 17 | +3.3V | 18 | +3.3V |
| 19 | GND | 20 | GND |

## Bill of Materials

| Ref | Value | Package | Description |
|-----|-------|---------|-------------|
| U1, U2 | 74LVC245A | TSSOP-20 | 8-bit level shifter, 5V tolerant |
| C1, C2 | 100nF | 0402 | Bypass capacitors |
| J1 | DIP-48 | Through-hole | Pass-through (socket + header pins) |
| J2 | 2x10 header | 2.54mm pitch | FPGA connection |

### 3.3V Power Supply Options
The 74LVC245s need 3.3V. Options:
1. **From FPGA board**: Supply 3.3V via J2 pins 17/18 from the Tang Primer 20K's 3.3V rail
2. **On-board LDO**: Add an AMS1117-3.3 regulator powered from DENISE's 5V (pin 19). Add input (10uF) and output (10uF) capacitors.

## PCB Layout Notes

- **DIP-48 footprint**: Use turned-pin (machined) headers for the bottom side (plugs into motherboard). Use a precision DIP socket on top (DENISE plugs in).
- **Pin spacing**: DIP-48 has 2.54mm pin pitch, 15.24mm row spacing, pins numbered counter-clockwise (1-24 on left, 25-48 on right going up).
- **Keep traces short**: Especially for 7M and CDAC clock signals. These run at ~7MHz but are used as PLL references.
- **Ground plane**: Use a ground pour on the bottom layer for signal integrity.
- **Board dimensions**: Approximately 70mm x 35mm fits the DIP-48 footprint plus level shifters and connector.
